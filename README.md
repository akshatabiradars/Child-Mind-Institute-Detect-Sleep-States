## Detect Sleep States Machine Learning Model
This repository contains the code and resources for a machine learning model developed to detect sleep states in children. The model was created as a part of the project for the Child Mind Institute to aid in sleep analysis and research.

## Features
Data Preprocessing: Includes scripts and notebooks for preprocessing raw sleep data.
Model Development: Implementation of machine learning models for sleep state classification.
Evaluation: Metrics and tools to evaluate the performance of the model.
Inference: Scripts to run the trained model on new data for sleep state prediction.
